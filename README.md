# 勇吉屋 BASEリニューアル確認用プレビュー

これはBASE本番へ適用する前の確認用プレビューです。

- `index.html`: BASE HTML編集 Appへの移設を想定したメインプレビュー
- `base-theme-migration-preview.html`: `index.html` と同じ内容の控え
- `contact.html`: お問い合わせページの確認用プレビュー
- `cart.html`: カートページの確認用プレビュー

商品画像は公開中の商品画像URLを参照しています。
学校名や学校用品を選ぶと、メインプレビュー内の検索結果がリアルタイムに切り替わります。
商品は1ページ最大50件ずつ表示し、ページめくりで確認します。
商品画像または「サイズを選ぶ」を押すと、BASE下書きと同じ商品詳細が開きます。
商品詳細ではCSVに登録された種類・サイズと数量を選び、確認用カートへ追加できます。
在庫のない商品は画像から開けず、グレーの「在庫なし」ボタンを表示します。
ヘッダーには `TOP / 店舗について / お問い合わせ / カート` の導線を置いています。
お問い合わせは新デザイン内で入力と内容確認まで動作し、確認中は実送信を止めています。
カートはプレビュー内で商品追加、数量変更、削除、概算合計まで確認できます。
下部にお支払い方法の案内とgpt-image2生成の説明ビジュアルを掲載しています。
店舗欄には、採寸予約と営業日を公式HPで確認する案内バナーを掲載しています。
スマホは端末の実幅に追従し、320pxを含む各幅でページ全体の横スクロールが出ないようにしています。
日本語は禁則処理とフレーズ単位の自動改行を使い、単語途中の不自然な改行を避けています。
説明画像・商品画像はタップまたはキーボード操作で拡大でき、スマホの店舗画像は2枚を小さく並べて表示します。
学校カテゴリページや商品ページなど、BASE本番側へ遷移する確認用リンクは置いていません。
商品表示の補助データは公開商品カードと種類・サイズ・在庫有無に限定し、在庫数、CSV列名、顧客・注文情報は含めません。
商品全件チェックはローカル確認用に限定し、GitHub Pagesには公開しません。
注文、在庫、顧客情報、BASE本番テーマは変更しません。

最新版の生成と同期:

```sh
node ../base_redesign/scripts/render-base-theme-local-preview.mjs
node ../base_redesign/scripts/sync-github-pages-preview.mjs
```
